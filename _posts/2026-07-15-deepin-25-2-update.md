Halo pengguna **deepin**! deepin 25.2.0 telah resmi dirilis. Update ini membawa berbagai peningkatan performa, perbaikan bug, dan fitur baru yang membuat pengalaman menggunakan deepin semakin lancar.

## 🚀 Apa yang Baru di deepin 25.2.0?

### Peningkatan DDE (Deepin Desktop Environment)
- **Performa lebih responsif** — animasi lebih halus dan penggunaan memori lebih rendah
- **Control Center** — panel pengaturan yang didesain ulang dengan navigasi yang lebih intuitif
- **Dock** — mode baru dan plugin yang dapat disesuaikan

### Peningkatan Sistem
- **Kernel Linux** — update ke versi terbaru untuk dukungan hardware yang lebih luas
- **Driver** — peningkatan kompatibilitas dengan GPU Intel, AMD, dan NVIDIA terbaru
- **Keamanan** — patch keamanan terbaru diterapkan

### Perbaikan Bug
- Memperbaiki crash pada File Manager saat membuka folder besar
- Menyelesaikan masalah audio Bluetooth pada beberapa perangkat
- Meningkatkan stabilitas koneksi Wi-Fi

## 📥 Cara Update

Buka **Control Center** → **Update** dan klik **Check for Updates**. Alternatifnya, gunakan terminal:

```bash
sudo apt update && sudo apt upgrade
```
