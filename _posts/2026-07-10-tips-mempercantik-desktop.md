Desktop **deepin** sudah cantik dari bawaannya, tapi kamu bisa membuatnya semakin personal dan keren dengan beberapa trik mudah. Yuk, simak!

## 1. 🎨 Ganti Tema Global

deepin punya banyak tema bawaan. Buka **Control Center** → **Personalization** → **Theme**. Coba tema **deepin** (terang) atau **deepin-dark** (gelap). Kamu juga bisa install tema komunitas!

## 2. 🖼️ Wallpaper Dinamis

Gunakan folder gambar sebagai source wallpaper yang berganti otomatis. Klik kanan desktop → **Set Wallpaper** → pilih folder berisi koleksi wallpapermu.

## 3. 📊 Tambahkan Widget

Klik kanan desktop → **Add Widget**. deepin punya widget seperti jam, kalender, CPU monitor, dan notes. Letakkan sesuai selera!

## 4. 🔧 Kustomisasi Dock

Di **Control Center** → **Personalization** → **Dock**, kamu bisa:
- Pindahkan posisi dock (bawah/kiri/kanan)
- Atur ukuran ikon
- Pilih mode **Fashion** atau **Efficient**

## 5. 🖥️ Efek Desktop

Aktifkan **Window Effect** di Control Center untuk animasi jendela yang lebih keren. Transparansi, blur, dan animasi minimize/maximize bisa diatur sesuai selera.
